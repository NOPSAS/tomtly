Bygg og deploy tomtly.no til produksjon.

1. Kjør `npm run build` i /c/Users/jakob/tomtly/
2. Hvis bygget feiler, fiks feilene og bygg på nytt
3. Kjør `vercel --prod --yes` for å deploye
4. Verifiser at deploy var vellykket (sjekk at output viser "Aliased: https://tomtly.no")
5. Rapporter tilbake med deploy-URL

Ikke spør om bekreftelse - bare bygg og deploy.
